# Houdini HDAs

Public Houdini Digital Asset (HDA) collection by AJ Haworth.

## Projects

### [Immersive Optimization Toolkit](./wwdc-optimization/)
A modular toolkit for real-time asset optimization in immersive environments.
16 HDAs for geometry optimization, visibility culling, UV projection, and more.

## Installation

1. Download or clone this repository
2. Copy desired HDAs to your Houdini `otls` directory
   - macOS: `~/Library/Preferences/houdini/XX.X/otls/`
   - Windows: `%USERPROFILE%/Documents/houdiniXX.X/otls/`
3. Restart Houdini or use **File > Refresh Asset Libraries**
